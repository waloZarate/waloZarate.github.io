    Descrición de los datos¶

    • Age: edad del beneficiario principal

    • sex: sexo del contratista de seguros, mujer, hombre

    • BMI: ndice de masa corporal, que proporciona una comprensin del cuerpo, pesos relativamente altos o bajos en relacin con la altura, ndice objetivo de peso corporal (kg / m ^ 2) utilizando la relacin entre la altura y el peso, idealmente 18,5 a 24,9

    • Children: Nmero de nios cubiertos por seguro mdico / Nmero de dependientes.

    • Smoker: Fumar

    • Region: rea residencial del beneficiario en los EE. UU., Noreste, sureste, suroeste, noroeste.

    • Charges: Costos mdicos individuales facturados por el seguro mdico.

    def calculate_accuracy(actual, predicted):
    SST = 0
    SSR = 0
    SSE = 0
    RMSE = 0
    VIF = 0
    RSqr = 0
    MAE = 0
    MAPE = 0
    SST = sum((actual - np.mean(predicted))**2)    # Calculate the SST
    SSR = sum((predicted - np.mean(predicted))**2) # Calculate the SSR
    SSE = sum((actual - predicted)**2)             # Calculate the SSE
    RMSE = np.sqrt((sum((predicted - actual)**2))/len(predicted))  # Calculate the RMSE
    RSqr = 1 - (SSE/SST)                           # Calcualte the R_square
    if RSqr != 1:
        VIF = 1 / (1 - RSqr)                           # Calculate the VIF
    #MAPE_house_price = mape(dtc_predict_train, y_train)           # Calculate the MAPE
    #MAE_house_price = mae(y_train, dtc_predict_train)             # Calculate the MAE
    return RMSE, RSqr, VIF
